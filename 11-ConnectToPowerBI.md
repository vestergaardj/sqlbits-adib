## Connect Azure Synapse Analytics to Power BI
造造造造 Make sure you publish the data set before proceeding

Click *Manage* in Synapse Studio

Select *Linked Services*

Click *New*

Select *Power BI*

Click *Continue*

Rename to: CrimeWorkspace

Select *Tenant*

Select *Workspace Name*

Click *Create*

Click *Publish All*

Click *Publish*
